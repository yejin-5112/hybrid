레포트: 반응형 웹 UX 개선

개요

사용자 경험 향상을 위해 기존 웹사이트의 반응형 디자인을 분석하고 개선 방안을 제시했습니다.

레이아웃 최적화

모바일과 데스크탑 화면에서 콘텐츠 간격과 가독성을 개선했습니다.

네비게이션 개선

메뉴 구조를 단순화하여 사용자가 원하는 기능을 빠르게 찾을 수 있도록 수정했습니다.

결론 & 느낀 점

UX 개선 후 사용자 만족도가 향상되었으며, 반응형 디자인의 중요성을 다시 한번 느꼈습니다.

프로젝트: 학사관리 앱

개요

학생 정보, 출석, 성적 관리를 통합한 웹 기반 학사관리 앱 개발 프로젝트입니다.

목표 & 역할

  • 목표: 사용자 친화적 UI 제공과 효율적 데이터 관리
  • 역할: 기획, 디자인, 프론트엔드, 백엔드
  • 기술 스택: HTML, CSS, JS, React, Node.js, MongoDB

주요 기능

  • 학생 정보 등록/수정/삭제
  • 출석 및 성적 관리
  • REST API 연동
  • 반응형 웹 디자인 지원

개발 과정 & 문제 해결

  • 기획 → 디자인 → 프론트엔드/백엔드 개발 → 테스트
  • REST API 연동 데이터 충돌 → 서버 검증 로직 추가로 해결
  • 모바일 레이아웃 깨짐 → CSS 미디어쿼리 최적화

결과 & 느낀 점

프로젝트 구현 완료 및 로컬 배포 완료

배운 점: 프론트엔드/백엔드 통합 경험, REST API 설계 경험, 반응형 웹 구현 경험
